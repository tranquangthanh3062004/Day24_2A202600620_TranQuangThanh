# Case Study 1: CNET - AI sinh bài viết tài chính sai lệch

## Brief Case
| Field | Bạn điền gì |
| :--- | :--- |
| **Tên case** | CNET AI Errors (2023) |
| **Ngành** | Media / news / social / political assistant |
| **Tổ chức / sản phẩm** | CNET / Hệ thống AI tạo nội dung nội bộ |
| **Use case AI** | Tự động tạo các bài báo giải thích kiến thức tài chính cơ bản để tối ưu SEO. |
| **Thời điểm** | Tháng 1/2023 |
| **Case xảy ra chuyện gì?** | CNET dùng AI viết bài và xuất bản dưới bút danh "CNET Money Staff". Trang Futurism phát hiện các bài báo này sai những kiến thức toán học, tài chính sơ đẳng (VD: Tính sai tiền lãi suất). |
| **Stakeholder bị ảnh hưởng** | Độc giả cá nhân (đọc sai kiến thức tài chính), Phóng viên CNET. |
| **Số liệu chính** | CNET đã phát hành **77** bài báo viết bằng AI. Trong đó **41 bài (hơn 53%)** phải đính chính do lỗi sai nghiêm trọng hoặc lỗi đạo văn. |
| **Trích nguồn ngắn** | Futurism phát hiện lỗi sai, buộc CNET phải thừa nhận 41/77 bài viết bằng AI cần đính chính lớn. |
| **Nguồn 1** | Futurism (11/01/2023) - [CNET Is Quietly Publishing Entire Articles Generated By AI](https://futurism.com/cnet-publishing-articles-by-ai) |
| **Nguồn 2** | The Verge (25/01/2023) - [CNET’s AI journalist committed extensive plagiarism](https://www.theverge.com/2023/1/23/23567675/cnet-ai-journalist-plagiarism-errors) |
| **Ghi chú độ tin cậy** | Secondary (High-quality investigative tech journalism). Không có conflict. |
